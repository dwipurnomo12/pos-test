## Installation & Setup

Follow these steps to get the project up and running on your local machine:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/dwipurnomo12/pos-test
    cd [pos-test]
    ```

2.  **Install PHP dependencies:**

    ```bash
    composer install
    ```
